# 게임 플레이 페이지 (/play/[id])

> 테스트에 사용할 챌린지 ID는 실행 시 `/challenges` 페이지에서 첫 번째 카드의 ID를 동적으로 가져온다.

## TC-1: 페이지 로드 & Idle 화면

1. `/play/{id}` 페이지에 접속한다
2. 챌린지 제목이 표시되는지 확인한다
3. 난이도 정보(쉬움/보통/어려움)가 표시되는지 확인한다
4. "시작하기" 버튼이 보이는지 확인한다
5. "속도" 버튼이 보이는지 확인한다

## TC-2: 속도 선택 모달

1. "속도" 버튼을 클릭한다
2. 속도 선택 모달이 열리는지 확인한다 ("속도 선택" 텍스트가 보이는지)
3. 슬라이더가 보이는지 확인한다
4. "확인" 버튼이 보이는지 확인한다
5. "확인"을 클릭하면 모달이 닫히는지 확인한다

## TC-3: 게임 시작 → 카운트다운

1. "시작하기" 버튼을 클릭한다
2. 카운트다운 화면이 나타나는지 확인한다 (숫자 3, 2, 1이 순서대로 표시)

## TC-4: 네비게이션 바

1. Idle 화면에서 "목록" 버튼이 보이는지 확인한다
2. "목록" 버튼을 클릭한다 → `/challenges`로 이동하는지 확인한다
