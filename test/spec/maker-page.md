# 챌린지 만들기 페이지 (/maker)

## TC-1: 페이지 로드 & 기본 렌더링

1. `/maker` 페이지에 접속한다
2. "이미지 관리" 섹션이 보이는지 확인한다
3. "라운드 1 / 5" 표시가 보이는지 확인한다
4. 8개 슬롯이 보이는지 확인한다
5. "생성하기" 버튼이 보이는지 확인한다

## TC-2: 라운드 수 변경

1. 라운드 수 "+" 버튼을 클릭한다 → "라운드 1 / 6"으로 변경되는지 확인한다
2. "-" 버튼을 3번 클릭한다 → "라운드 1 / 3"으로 변경되는지 확인한다
3. "-" 버튼을 계속 클릭하여 1까지 줄인다 → "-" 버튼이 비활성화되는지 확인한다
4. "+" 버튼을 클릭하여 5까지 올린다 → "+" 버튼이 아직 활성화인지 확인한다 (최대 5)

## TC-3: 라운드 네비게이션

1. 라운드 수를 3으로 설정한다
2. "다음" 버튼(Round 네비게이션)을 클릭한다 → "라운드 2 / 3"이 되는지 확인한다
3. 다시 "다음"을 클릭한다 → "라운드 3 / 3"이 되고 "다음" 버튼이 비활성화되는지 확인한다
4. "이전" 버튼을 클릭한다 → "라운드 2 / 3"으로 돌아가는지 확인한다

## TC-4: 이름 표시 토글

1. "이름 표시" 토글이 보이는지 확인한다
2. 토글을 클릭하여 상태가 변경되는지 확인한다
