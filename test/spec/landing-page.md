# 랜딩 페이지 (/)

## TC-1: 히어로 섹션 렌더링

1. `/` 페이지에 접속한다
2. "단어리듬게임" 제목이 보이는지 확인한다
3. "Say The Word On Beat" 부제목이 보이는지 확인한다
4. 공개 챌린지 수와 총 플레이 수가 표시되는지 확인한다
5. "나만의 게임 만들기" 버튼이 보이는지 확인한다
6. "인기 챌린지 플레이" 버튼이 보이는지 확인한다

## TC-2: CTA 버튼 동작

1. "인기 챌린지 플레이" 버튼을 클릭한다 → `/challenges` 페이지로 이동하는지 확인한다
2. 뒤로가기 후 "나만의 게임 만들기" 버튼을 클릭한다 → 모달이 열리는지 확인한다 ("시작하기" 또는 "공개/비공개" 텍스트가 보이는지)

## TC-3: 인기 챌린지 피드

1. "인기 챌린지" 섹션이 보이는지 확인한다
2. 챌린지 카드가 1개 이상 표시되는지 확인한다
3. "모든 챌린지 보기" 링크를 클릭한다 → `/challenges`로 이동하는지 확인한다

## TC-4: 피드 챌린지 카드 클릭

1. 인기 챌린지 섹션의 첫 번째 카드를 클릭한다
2. `/play/{ID}` 형태의 플레이 페이지로 이동하는지 확인한다
