# 라운드 수 유연화 (Flexible Round Count)

**상태**: 계획 중
**기준선 측정일**: 2026-04-05 (GA4 30일 데이터)

## 배경

### 데이터 근거

**GA4 퍼널 (30일)**:
- game_start: 15,218건 (2,697명)
- round_complete: 49,462건 → **게임당 평균 3.25라운드** (5라운드 중)
- game_complete: 9,025건 → **완료율 59.3%**
- **40.7%가 중도 이탈** — 5라운드를 다 못 채움

**설문 피드백 (교사)**:
- "5장은 적어요. 10장 이상이면 좋겠습니다"
- "부분별로 끊어서 할 수 있으면 좋겠어요"

**크리에이터 퍼널**:
- challenge_create_start → complete: **51.9%** 완료율
- 5라운드 × 8슬롯 = 40장의 이미지 배치 부담

### 문제 정의

1. **플레이어**: 5라운드가 길어서 40%가 중도 이탈
2. **크리에이터**: 5라운드 고정이라 콘텐츠 준비 부담이 커서 48%가 생성 포기
3. **유연성 부재**: 짧은 퀴즈(1-2라운드)도, 긴 챌린지(10라운드)도 불가

## 가설

> 라운드 수를 1~10으로 유연하게 선택할 수 있으면:
> - 짧은 챌린지(1-3라운드)의 완료율이 80%+로 상승하고
> - 크리에이터 생성 완료율이 60%+로 상승하며
> - 전체 챌린지 수가 증가하여 콘텐츠 다양성이 개선된다

## 목표 지표 (North Star)

**게임 완료율** = game_complete / game_start

- 기준선: 59.3%
- 목표: **70%+** (2주 후)

이유: 라운드 유연화의 핵심 효과는 "끝까지 플레이하는 비율 증가"이므로 완료율이 가장 직접적인 성공 척도.

## 추적 이벤트

### 기존 이벤트 (변경 없음)

| 이벤트 | 용도 |
|--------|------|
| `game_start` | 퍼널 시작 |
| `round_complete` | 라운드별 진행 추적 |
| `game_complete` | 퍼널 완료 |
| `game_replay` | 재도전 |
| `challenge_create_start` | 생성 퍼널 시작 |
| `challenge_create_complete` | 생성 퍼널 완료 |

### 신규 이벤트

| 이벤트 | 파라미터 | 용도 |
|--------|----------|------|
| `game_start` (파라미터 추가) | + `round_count` | 게임의 총 라운드 수 |
| `game_complete` (파라미터 추가) | + `round_count` | 완료된 게임의 라운드 수 |
| `challenge_create_complete` (파라미터 추가) | + `round_count` | 생성된 챌린지의 라운드 수 |

기존 이벤트에 `round_count` 파라미터만 추가하면 됨. 새 이벤트 불필요.

### 핵심 분석 쿼리

1. **라운드 수별 완료율**: `game_complete(round_count=N)` / `game_start(round_count=N)`
2. **라운드 수 분포 (플레이)**: `game_start` 이벤트의 `round_count` 분포
3. **라운드 수 분포 (생성)**: `challenge_create_complete`의 `round_count` 분포
4. **크리에이터 전환율 변화**: 출시 전후 `create_start → create_complete` 비율 비교
5. **짧은 챌린지 재도전율**: `round_count ≤ 3`인 게임의 `game_replay` 비율
